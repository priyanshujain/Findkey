Welcome

This is homepage for project Findkey. This is a small game project but we are building it as an open source project.

We are using it to record drafts of our tasks and results. Welcome abroad! This is aimed at people starting to contribute to Findkey.

About

This game is about finding keys and opening gates, as player find key of a color he is able to open that colored gate and there is a grand door that can be opened only when he has grand key and at a time player can hold only one key. When he is able to open grand door then it becomes door for the next level.
